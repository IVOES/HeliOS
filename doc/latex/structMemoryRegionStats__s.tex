\doxysubsubsection{Detailed Description}
The Memory\+Region\+Stats\+\_\+t data structure is used by \mbox{\hyperlink{HeliOS_8h_a96142397ee8d9044d76e8728c4d3ca9c}{x\+Mem\+Get\+Heap\+Stats()}} and \mbox{\hyperlink{HeliOS_8h_a0fa09bf6e80a31e914323e006021416a}{x\+Mem\+Get\+Kernel\+Stats()}} to obtain statistics about either memory region. The Memory\+Region\+Stats\+\_\+t type should be declared as x\+Memory\+Region\+Stats.
