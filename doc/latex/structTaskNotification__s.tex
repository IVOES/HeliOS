\doxysubsubsection{Detailed Description}
The Task\+Notification\+\_\+t data structure is used by \mbox{\hyperlink{HeliOS_8h_a6d6720b61c53a25f1435e2244f695952}{x\+Task\+Notify\+Give()}} and \mbox{\hyperlink{HeliOS_8h_abe3e24aeda10f58c64d3c4f07b656248}{x\+Task\+Notify\+Take()}} to send and receive direct to task notifications. Direct to task notifications are part of the event-\/driven multitasking model. A direct to task notifcation may be received by event-\/driven and co-\/operative tasks alike. However, the benefit of direct to task notifications may only be realized by tasks scheduled as event-\/driven. In order to wait for a direct to task notification, the task must be in a \char`\"{}waiting\char`\"{} state which is set by \mbox{\hyperlink{HeliOS_8h_aea800988094ef870f58a4303afe82625}{x\+Task\+Wait()}}. The Task\+Notification\+\_\+t type should be declared as x\+Task\+Notification.
