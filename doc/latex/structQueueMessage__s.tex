\doxysubsubsection{Detailed Description}
The Queue\+Message\+\_\+t stucture is used to store a queue message and is returned by \mbox{\hyperlink{HeliOS_8h_a64f2a699dbbe6e9ca1131633522fc799}{x\+Queue\+Receive()}} and \mbox{\hyperlink{HeliOS_8h_a0183c622db0b9ac27c8d2069783115e2}{x\+Queue\+Peek()}}. The Queue\+Message\+\_\+t stucture should be declared as x\+Queue\+Message.

\begin{DoxyAttention}{Attention}
The memory allocated for the data structure must be freed by calling \mbox{\hyperlink{HeliOS_8h_a0d4765ea27e4a15d0b944ce62db5f059}{x\+Mem\+Free()}}.

The message value is {\itshape NOT} null terminated and thus Standard C Library string functions such as strcmp(), strcpy() and strlen(), which expect a null terminated char array, must not be used to manipulate the message value.
